Portafolio

Proyecto gestión de facturas

El proyecto se basa en la creación y proceso de facturas en una empresa, donde recorre distintas áreas, con las respectivas fechas y estados necesarios para cada área.

Desarrollado en el Framework Ruby On Rails, Base de Datos Mysql, Repositorio en Github por medio de Git en la terminal de Ubuntu.

Sobre Mi

La pasión que tengo por la tecnología es algo que me ha caracterizado desde muy pequeño, y ahora que decidí entrar a este mundo, supe que no sería fácil, pero las veces que me he propuesto aprender e intentar lo he logrado.

Mi experiencia actual es de nivel técnico, con conocimiento en C#, Java, HTML, CSS, JavaScript, bases de datos relacionales como MySQL y PostgreSQL, sistema operativo Ubuntu, Git, Github, pero mi experiencia más sólida es en el lenguaje de Ruby con el framework Ruby On Rails.
